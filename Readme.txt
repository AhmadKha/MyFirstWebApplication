Damit die Pfade relativ zur Index.html gleich bleiben muss die unterliegende Ordnerstruktur beim Herunterladen von Github beibehalten werden. 

Zum Starten der Web-Applikation, Doppelklick auf die Datei index.html.
Ggf. kann dann ein beliebiger Browser (Chrome, Firefox, Internet Explorer) ausgesucht werden.

Die Seite reagiert, bei Klick auf den schwarzen Pfeil oder auf einen der Services.
