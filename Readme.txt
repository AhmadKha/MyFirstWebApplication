Damit die Pfade relativ zur Index.html gleich bleiben muss die Ordnerstruktur beim Herunterladen von Guthub beigehalten werden. 

Zum Starten der Web-Applikation, Doppelklick auf die Datei index.html ausführen.
Ggf. kann dann ein beliebiger Browser (Chrome, Firefox, Internet Explorer) ausgesucht werden.

Die Seite reagiert, wenn der man auf den schwarzen Pfeil oder auf einen der Services klickt.
